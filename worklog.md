## Task ID: container-rate-calculator
### Work Task
Создание веб-приложения для котировки ставок контейнерных автоперевозок по России

### Work Summary
Создано полноценное веб-приложение для расчёта ставок контейнерных автоперевозок со следующими возможностями:

**Созданные файлы:**
1. `/src/lib/data/constants.ts` - Константы расчёта (цена топлива 75 руб/л, коэффициенты контейнеров, режимов перевозки, классов опасности ADR, региональные коэффициенты)
2. `/src/lib/data/cities.ts` - База городов (45 городов России и Беларуси) с матрицей расстояний между основными логистическими хабами
3. `/src/lib/calculator.ts` - Модуль расчёта ставки с формулой себестоимости и рыночной вилкой
4. `/src/components/CitySelect.tsx` - Компонент выбора города с поиском и группировкой по регионам
5. `/src/components/RateForm.tsx` - Форма ввода параметров (маршрут, тип контейнера, вес, режим, опасность)
6. `/src/components/RateResult.tsx` - Компонент отображения результатов с детализацией расчёта
7. `/src/app/api/calculate/route.ts` - API endpoint для расчёта ставки
8. `/src/app/page.tsx` - Главная страница приложения

**Функционал:**
- Выбор города отправления и назначения с поиском (45 городов)
- 4 типа контейнеров: 20' DC, 20' HC, 40' DC, 40' HC
- Валидация веса груза по типу контейнера
- 3 режима перевозки: ГТД, ВТТ, МТТ
- 12 вариантов опасности груза (ADR классы 1-9 + общие категории)
- Расчёт себестоимости и ставки без НДС
- Рыночная вилка (от мин до макс)
- Детализация расчёта с коэффициентами
- Современный UI в синих/серых тонах
