# Настройка Google Sheets для расчёта расстояний

## Проблема
Текущий Web App URL не работает. Нужно заново развернуть Google Apps Script.

## Решение

### Шаг 1: Откройте Google таблицу
https://docs.google.com/spreadsheets/d/10dHttNty-QPAiHo0bjrktHtozcdf8HEPYe5dSkdf8SI/edit

### Шаг 2: Настройте структуру таблицы
Убедитесь, что в строке 1 (заголовки) следующие названия:

| A | B | C | D | E | F | G | H | I |
|---|---|---|---|---|---|---|---|---|
| НТ | ТЗП | ТЗ | КТ | ТСП | ДТ1 | ДТ2 | ДТ3 | Расстояние |

### Шаг 3: Добавьте формулу в ячейку I2
Вставьте в ячейку **I2** следующую формулу:

```
=GOOGLEMAPS_DISTANCE(A2;B2;C2;D2;E2;F2;G2;H2)
```

**Если формулы GOOGLEMAPS_DISTANCE не существует**, используйте встроенную:

```
=ЕСЛИ(ИЛИ(C2="";D2="");0;ОКРУГЛ(MAPs!A1/1000))
```

Или создайте пользовательскую функцию (см. ниже).

### Шаг 4: Откройте Apps Script
1. В меню: **Расширения** → **Apps Script**
2. Удалите весь существующий код
3. Вставьте код из файла `google-apps-script.js`

### Шаг 5: Сохраните и разверните
1. Нажмите **Ctrl+S** (или Cmd+S) для сохранения
2. Нажмите **Deploy** → **New deployment**
3. Нажмите на значок шестерёнки и выберите **Web app**
4. Заполните:
   - **Description**: Distance Calculator
   - **Execute as**: Me
   - **Who has access**: **Anyone** (важно!)
5. Нажмите **Deploy**
6. **Скопируйте новый URL** и пришлите его мне

### Шаг 6: Авторизация
При первом развертывании Google попросит авторизовать скрипт:
1. Нажмите "Review permissions"
2. Выберите ваш Google аккаунт
3. Нажмите "Advanced" → "Go to ... (unsafe)"
4. Нажмите "Allow"

## Альтернативное решение (проще)

Если формула Google Maps не работает, я добавил в скрипт прямой расчёт через Maps API. Скрипт автоматически использует его как резервный вариант.

## Формула для I2 (без пользовательской функции)

Если пользовательская функция не работает, можно использовать такую формулу:

```
=ЕСЛИ(ИЛИ(C2="";D2="");"";IMPORTXML("https://maps.googleapis.com/maps/api/distancematrix/xml?origins="&ENCODEURL(C2)&"&destinations="&ENCODEURL(D2)&"&key=ВАШ_API_KEY";"/DistanceMatrixResponse/row/element/distance/value"))/1000)
```

Но это требует API ключа Google Maps.
